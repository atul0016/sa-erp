## Long Description
SA ERP is a comprehensive, production-ready Enterprise Resource Planning system purpose-built for Indian businesses. Developed using React 18, TypeScript, Tailwind CSS, and Electron 30, it features 10 fully integrated modules covering Finance & Accounting (double-entry with bank reconciliation), Inventory (multi-warehouse, FIFO/WA, gate passes), Sales (POS terminal, credit limits), Purchase (approval workflows), Manufacturing (MRP, Shop Floor Kiosk, OEE Dashboard), HRM & Payroll (PF/ESI/TDS), GST Compliance (E-Invoice, E-Way Bill, GSTR-1/3B), TDS Management (Form 26Q), and 16 pre-built report templates.

The system implements MCA-compliant SHA-256 chained audit trails, role-based access control, multi-tenant isolation, and covers every major Indian compliance requirement — GST, TDS, PF, ESI, Professional Tax, Form 16, GSTIN/HSN validation. It runs as a web app (deployed on Vercel) and a native desktop app (via Electron Builder) from a single codebase, with an SQLite database powering 80+ tables.
